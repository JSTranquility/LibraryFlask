API Pública

Endpoints públicos del catálogo Vellum. No requieren autenticación.

GET /api/books

Lista completa de todos los libros.

curl http://127.0.0.1:5000/api/books
GET /api/books/<id>

Detalle de un libro por ID.

curl http://127.0.0.1:5000/api/books/1
GET /api/books/paginated?page=1&per_page=12&genre=&search=&sort=newest

Catálogo paginado con filtros opcionales.

pageintdefault: 1
per_pageintdefault: 12
searchstringbúsqueda por título o autor
genrestringfiltrar por género
languagestringfiltrar por idioma
yearstringfiltrar por año
sortstringnewest, oldest, az
curl "http://127.0.0.1:5000/api/books/paginated?page=1&per_page=5&genre=Ficción"
GET /api/books/<id>/reviews

Reseñas de un libro.

curl http://127.0.0.1:5000/api/books/1/reviews
GET /api/books/fetch-by-isbn?isbn=978...

Busca metadata de un libro por ISBN (Open Library).

curl "http://127.0.0.1:5000/api/books/fetch-by-isbn?isbn=9780061120084"
GET /api/stats/reviews

Total de reseñas en la plataforma.

curl http://127.0.0.1:5000/api/stats/reviews
